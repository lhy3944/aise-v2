# 작업 동기화

아래 단계를 수행하세요:

1. **PLAN.md**를 읽고 전체 작업 계획과 미완료 항목(`[ ]`)을 파악
2. **PROGRESS.md**를 읽고 현재 상태 테이블과 최근 작업 로그를 파악
3. **references/** 디렉토리의 파일 목록을 확인하여 참고 가능한 조사 자료 파악
4. 다음 정보를 요약하여 브리핑:
   - 현재까지 완료된 작업
   - 다음에 해야 할 작업 (우선순위 순)
   - 블로커나 미결정 사항 (TBD 항목 등)
   - 참고할 수 있는 레퍼런스 자료
5. 사용자에게 다음 작업을 제안
