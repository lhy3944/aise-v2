# 작업 계획 확인/업데이트

**PLAN.md**를 읽고 다음을 수행하세요:

## 인자가 없거나 "check"인 경우
- 전체 Phase별 진행률을 보여줌 (완료/전체)
- 미완료 항목(`[ ]`)을 우선순위 순으로 나열
- 다음에 착수할 작업을 제안

## "update"인 경우
- 사용자에게 완료된 항목을 확인
- PLAN.md에서 해당 항목을 `[x]`로 체크
- PROGRESS.md도 함께 업데이트

인자: $ARGUMENTS
